## Plan of the project
[russian below | русское описание ниже]

The aim of this project is to provide an interactive planner for your tasks and goals. The process of using the app is supposed to look like this:
1. The client signs up. While doing it, one mentions one's global goals (sports, studies, reading books, eco-activity, etc.). The list of goals can be edited later, so don't worry if your plans change. To use the application later, authorization will be required.
2. When entering the first page after authorization, one sees the day's schedule. The day's schedule can be filled with activities (indicate start time and expected duration). Each activity can be matched with a goal from one's global list of goals.
 It is optional.
3. The main feature of this app is storing statistics about your activities! All the schedules will be stored in a database, so one will be able to monitor the process of achieving the goal.
The  following types of statistics are planned to be supported:
    * Week/month statistic: total amount of time spent on each goal
    * Goal statistic per week, month, or year: time spent for this particular goal distribution by days, weeks, or months
    * Retrieving the day schedule for a particular day within the last year
  

[Описание на русском языке]
Цель этого проекта - предоставить интерактивный планировщик для ваших целей и задач. Предполагаемый процесс использования приложения таков:
1. Клиент регистрируется в приложении. Во время этого он указвает список своих глобальных целей (спорт, учеба, чтение книг, эко-активизм и т.п.). Этот список можно редактировать позже, поэтому не волнуйтесь, если планы изменятся :).
   Для дальнейшего использования приложения потребуется авторизация.
2. На первой странице (после авторизации) можно увидеть распорядок дня. Его можно заполнить задачами (активностями), указывая время начала и предполагаемую продолжительность. Каждую задачу можно связать с целью из глобального списка задач. Но это опционально.
3. Главная черта этого приложения - сохранение статистики о ваших занятиях! Все распорядки дня сохраняются в базе данных, поэтому вы сможете отслеживать прогресс в достижении своих целей.
   Планируется поддержать следующие виды статистики:
    * Статистика недели/месяца: суммарное количество времени за неделю/месяц, потраченное на достижение каждой из целей
    * Статистика цели за неделю, месяц или год: показывает распределение времени, потраченного на достижение конкретной цели по дням, неделям или месяцам
    * Вывод распорядка дня за указанный день (только за последний год)
